Este projeto tem como objetivo extrair dados de usuários a partir de uma API e armazená-los inicialmente em sua forma bruta, seguindo o conceito da camada Bronze em arquiteturas modernas de dados. Nessa etapa, os dados são preservados o mais próximo possível de sua forma original, permitindo rastreabilidade e reprocessamento caso seja necessário no futuro.

Na camada Silver, os dados passam por processos de limpeza, padronização e enriquecimento. Informações são transformadas para formatos adequados, campos são estruturados e novos atributos são derivados para melhorar a qualidade e a utilidade do conjunto de dados. Essa etapa garante que os dados estejam consistentes e prontos para análises posteriores.

Por fim, na camada Gold, o projeto disponibiliza um produto de dados voltado para um time de marketing que deseja, partir dos dados tratados, produzir análises de clientes por grupo etário e por região geográfica. Essas informações ajudam a entender melhor o perfil dos usuários e podem apoiar estratégias de segmentação e planejamento de campanhas.

Este projeto foi desenvolvido no âmbito da plataforma Databricks e o arquivo .DBC se encontra disponível neste repositório.
